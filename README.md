# Career Path Recommender System

## Overview
This project focuses on building a machine learning-based Career Path Recommender System that provides personalized career recommendations based on users' skills, interests, and career preferences.

The system explores unsupervised learning and recommendation techniques to identify patterns in user profiles and suggest relevant career paths, learning resources, and skill development opportunities aligned with market trends.

---

## Features
- Skill-based career recommendations
- User profile clustering
- Personalized recommendation generation
- Collaborative filtering concepts
- Career trend analysis and visualization
- Exploration of real-time market alignment

---

## Machine Learning Techniques Used
- K-Means Clustering
- DBSCAN
- Collaborative Filtering
- PCA (Dimensionality Reduction)

---

## Tech Stack
- Python
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- Tableau
- GeoPandas

## Workflow
1. User Data Collection
2. Skill and Interest Analysis
3. User Clustering using Unsupervised Learning
4. Recommendation Generation
5. Market Trend Analysis
6. Evaluation and Visualization
